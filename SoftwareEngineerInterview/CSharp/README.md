# CSharp Project


## Requirements
```
Microsoft Visual Studio Community 2022 (64-bit) > Version 17.4.2
.Net Core 3.1 (LTS)
```

## Install
```
cd Zip.InstallmentsService
dotnet restore
```

## Quick Start
```
cd Zip.InstallmentsService
dotnet run
```

## Run Tests
```
cd Zip.InstallmentsService
dotnet test Zip.InstallmentsService.sln
```
